# 📊 E-commerce Sales Dashboard – Madhav Store

## 📌 Project Overview
This project analyzes e-commerce sales data of Madhav Store using Power BI.  
The goal is to generate insights on sales, profit, customer trends, and payment modes to support business decisions.

## 🛠️ Tools & Technologies
- Power BI
- Data Cleaning & Transformation
- Data Visualization & Dashboard

-## 📈 Dashboard Features
- Sum of Amount by State (Sales Distribution)
- Sum of Quantity by Category
- Sum of Amount by Customer Name
- Payment Mode Analysis
- Monthly Profit Analysis
- Profit by Sub-Category

## 🔍 Key Insights
- Top customers contributing the highest revenue  
- Most popular product categories based on quantity sold  
- Preferred payment modes of customers  
- Monthly profit trends for business planning  
- High-profit and low-profit sub-categories identified
